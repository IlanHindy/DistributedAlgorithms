19  * Class AsynchronousSender
20  * This is a class which contains static methods to send one message
21  * The channel holds the sending socket
22  * The initialize value of the sending socket is null
23  * If the value of the sending socket is null (That meens that this is
24  * the first sending in the channel) a new socket is created and
25  * a connection to the receive process listening socket is made
26  * then the new socket is saved in the channel
27  * If the value of the sending process is not null the sending socket is
28  * retrieved from the channel
29  * Then a sending is done from the socket the termination character for the sending
30  * message is '#'
31  *
32  * The process of the connecting and the sending is as follows:
33  * 1. (Only if this is the first sending from the socket)
34  * The StartSending methods calls the BeginConnect method of the socket
35  * 2. When the socket created the connection it calls the ConnectCallback method
36  * 3. The ConnectCallback method calls the EndConnect method of the socket
37  * 4. After the connection done the StartSending method calls the Send method
38  * 5. The Send method calls the BeginSend method of the socket
39  * 6. When the socket finished the sending it calls SendCallback method
40  * 7. The SendCallback method calls the EndSend of the socket
41  *
42  * All the data needed for the method operation is passed in an instance of SocketStateObject
43  * Which is filled in StartSending and passed between the methods of this object and the
44  * socket's object. The AsynchronousSender class does not contain member variables because
45  * the methods are static and one instance serves all the processes
46  *
47  */
