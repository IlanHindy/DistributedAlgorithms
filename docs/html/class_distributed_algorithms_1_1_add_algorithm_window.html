Detailed Description

Dialog for creating or updating an algorithm automatic code data.

Description.
  • This class gets the following parameters:
    1. Source algorithm subject
    2. Source algorithm name
    3. Target algorithm subject
    4. Target algorithm name
  • This class produces the following:
    1. A code files for each one of the algorithm components:
      1. Network
      2. Process
      3. Channel
      4. Message
    2. A file for the parts of the classes that change (hopefully) only by the program The file contains:
      1. All the enums needed
      2. A message building methods for all the messages that will be used by the algorithm The methods are found in partial class of the process
      3. A partial class for the algorithm components which contains: the initializations of the algorithm's data structure
      4. Constants for accessing attributes by name
      5. Getters/setters to be used by the class
