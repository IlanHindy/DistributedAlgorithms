Attribute public members

  • Value (dynamic) - The value
  • Editable (bool) - Whether the attribute can be edited in the GUI after it was created
  • IncludeInShortDescription (bool) -
  • __EndInputOperationMethod - A method that is activated after the value was changed in the GUI For checking the value etc..
