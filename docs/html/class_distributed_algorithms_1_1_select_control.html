Detailed Description

A select control.

Description.
This control is used for selection from a list of strings
Usage Notes.
  • The following are the attributes of the control
    1. EnableTextBox
    2. SelectionMode
    3. Message
  • The following are the events of the control
    1. SelectionChanged in the list box
    2. NewItemCreated (When inserting value in the TextBox if the text is part of an existing item that item is selected. else a new item is opened and this event is fired)
