Distributed Algorithms
Creating the initial code

Preparations

  1. Specify if the network is a directed network and if the algorithm is centralized
  2. Decide what are the messages used by the algorithm and what are their components
  3. For each NetworkElement (Network, Process, Channel) decide :
    1. What are the parameters for the NetworkElement (Parameters are variables that do not change during execution)
    2. What are the variables for each NetworkElement (Variables can change during execution)

Create The initial code files

  1. Enter the program
  2. Select Program -> Add or Update Algorithm
  3. Select a source and target algorithm
    1. If this a new algorithm select the Template algorithm as source
    2. If this is an update to an algorithm select the algorithm to update as source
  4. In the dialog in the Messages item specify the messages that will be used in the algorithm
  5. In the dialog in the Network's ElementAttributes set the Centralized and the DirectedNetwork attributes
  6. In the dialog fill the parameters (PrivateAttributes) and the Variables (OperationResults) of the Network, Process, Channel
  7. Click the Create button and the code generated will appear and will be generated
  8. Exit the program
  9. There are 2 kinds of directories and files generated
    1. The code files in the Algorithms folder according to the config
    2. Data files in the Algorithm Data folder according to the config
  10. Include the new code files in the Visual Studio project

The following is a summary of the code generated:

Code generated in the InitAndDef files

Note :

The code of this file must not be change manually if you want to change the setting reopen the Add or Update Algorithm dialog

For messages:

Enums
  1. A main enum called MessageTypes that holds the possible messages
  2. An enum for each message holds it's components
  3. An enum for each dictionary which is a component of the message
Methods (The methods are part of the Process class because this is the class that uses them)
  1. A send method for each message
  2. A create method for each message that can be used to build the message
  3. A create method for each dictionary or list that are part of the message

For NetworkElements:

Enums (For each NetworkElement)
  1. Enum for each one of the 2 main dictionaries (PrivateAttributes, OperationResults)
  2. Enum for each one of the dictionaries that is a components of the variables or the parameters
Methods (For each one of the NetworkElements)
  1. An Init method for each one of the 2 dictionaries
  2. An Init method for each one of the dictionaries or lists that are components of the variables or parameters

Delete algorithm

  1. From the Visual Studio project perform Exclude From project to the algorithm code (Under Algorithms)
  2. From the Visual Studio project delete the folder of the algorithm from Algorithms
  3. From the Visual Studio project delete the folder of the algorithm from Algorithm Data
