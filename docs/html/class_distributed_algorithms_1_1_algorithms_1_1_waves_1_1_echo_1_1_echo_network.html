protected override void DistributedAlgorithms.Algorithms.Waves.Echo.EchoNetwork.CreateInitNetwork ( )
protected

Creates init network.

Description.
This method creates the initial network use this method if you want that the init operation (from the GUI) will create a network with some processes and channels
Algorithm.
Usage Notes.
The following is an example for how to create an init network (It is the method of the BaseNetwork)
protected virtual void CreateInitNetwork()
{
//Init the processes
int numberOfProcesses = 2;
int channelIdx = 0;
for (int idx = 0; idx &lt numberOfProcesses; idx++)
{
//Create and init a process
BaseProcess process = ClassFactory.GetProcess(this);
Processes.Add(process);
process.Init(idx);
//Create a Channel from the process to itself - used to terminate the sockets
//Used by the process
BaseChannel channel = ClassFactory.GetChannel(channelIdx, idx, idx, this);
Channels.Add(channel);
channelIdx++;
}
//Init the other channels
int numberOfChannels = 4;
for (; channelIdx &lt numberOfChannels; channelIdx++)
{
BaseChannel channel = ClassFactory.GetChannel(channelIdx, channelIdx % 2, (channelIdx + 1) % 2, this);
Channels.Add(channel);
}
}

A prototype for attribute checking.

Description.
Algorithm.
Usage Notes.
  1. Each Attribute can be assigned a value check that will be performed when changing the attribute from the network edit window while in design mode
  2. For each attribute that you want to add a check to, one method with the prototype of the following method have to be designed
  3. In order to add the check to the attribute you have to set the EndInputOperationMethod variable of the Attribute with the method you generated for example :
    // Setting the method to an existing attribute
    attribute.EndInputOperationMethod = EchonetworkDefaultAttributeCheck
    // Setting the method to an attribute while creation
    pa.Add(n.pak.SomeKey, new Attribute {Value = SomeValue, EndInputOperationMethod = EchonetworkDefaultAttributeCheck})

protected override void DistributedAlgorithms.Algorithms.Waves.Echo.EchoNetwork.CheckBuild ( List< BuildCorrectionParameters >  buildCorrectionsParameters)
protected

Check build.

Description.
This method performs checks on the network at the check network phase (before running and after designing)
checking network process.
The check and correct of a network element is composed from :
  1. A CheckBuild method that does the checks and for each error it finds it fills a BuildCorrectionParameters data structure and insert it to a list that is given as parameter to the method
  2. The BuildCorrectionParameters data structure is composed from :
    1. A message that will be presented
    2. A method that corrects the error (a method with the same prototype as the CorrectionMethod of this class)
    3. An object for passing the parameters from the check method to the correction method.
Algorithm.

Usage Notes.
  • The processes of the network are in network.Processes
  • The channels of the network are in network.Channels.

public void DistributedAlgorithms.Algorithms.Waves.Echo.EchoNetwork.CorrectionMethod ( object  correctionParameters)

Correct an error detected during check build.

Description.
  • This method is an example for the methods that needs to be created for correcting build errors.
  • For each error there has to be a method from this prototype that corrects the error.
Algorithm.
Usage Notes.
  1. Create a correction method for each error with a method with this prototype
  2. In the CheckBuild when encountering an error set the correction method name in the BuildCorrectionParameters data structures
  3. The methods gets its parameters from the object parameter. The structure of the parameters has to be the same in the CheckBuild method and the correction method
  4. The processes of the network are in network.Processes
  5. The channels of the network are in network.Channels.
