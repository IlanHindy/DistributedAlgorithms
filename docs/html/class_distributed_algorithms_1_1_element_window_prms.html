Detailed Description

An input field prms.

Description.
This class holds the specifications of the fields in ElementWindow that will be used for input, view and change the attribute in element input windows.
Usage Notes.
There are 3 fields for each attribute in the ElementWindow:
  1. The Header text box (which usually present the key of the attribute in the dictionary)
  2. The existing value text box
  3. The input control (can be TextBox, AddRemoveGrid, ComboBox) for changing the attribute value.
