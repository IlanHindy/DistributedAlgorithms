Checks activated before running

Introduction

  1. Before running the user has to press the check network button
  2. When this button is presses the following is done
  3. For each network element in the design (the network, the channels, the processes) the following is done
  4. If the network element was not editted or was changed :
    1. The CheckBuild method is activated. The CheckBuild method activates all the checks needed for the network element
      1. The CheckBuild method is filling a List that contains an entry for each check that failed
      2. The type of the entry is BuildCorrectionParameters and it includs 3 variables:
        1. A string with the message that will present when asking the user if he wants to correct the error
        2. The correction method
        3. An object for the parameters for the correction method
    2. For each error that was detected
      1. A dialog asking the user if he wants to correct the method is shows
      2. If the user answeres yes the correction method is activated
