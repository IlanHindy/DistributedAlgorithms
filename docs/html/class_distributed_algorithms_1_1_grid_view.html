Detailed Description

Interaction logic for GridView.xaml.

Description.
  • The GridView control is a control that presents a list of strings and allows Changing the order of the strings in the list
Usage Notes.
To use the control:
  1. In the object that the GridView present create a method with the signature of the ChangeFinished() delegate. This method should implement the order change. This method is called when the user finished the changes.
  2. Assign the method to the ChangeFinishedEvent() of this class
  3. Create a method with the signature of PresentPermutation() delegate. This method should create 2 strings one for the original list and one for the list after the transformation. This method is used to present the results of the order change for the end change dialog
  4. Pass this method is as a parameter to the constructor
  5. The GridView is created once and each time a modification is needed call ReplaceContent()
