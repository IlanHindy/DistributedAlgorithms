Detailed Description

Dialog for creating attributes.

Description.

The output this dialog is ;

  1. An attribute with all the fields
  2. The key for the attribute in the AttributeDictionary that it is ment to be added to

The process of using this dialog is

  1. Init : All possible values are inserted to members, The list box are fielded
  2. Selecting : Selecting from the list boxes
  3. Finish : Generate Attribute and fill it with the selection results

Usage Notes.

After the dialod was shoen there are to outputs to retrieve;

  1. newAttribute - The Attribute object if it is null it meens that the dialog quitted
  2. key - The key
