Detailed Description

Interaction logic for DebugWindow.xaml. This window Print messages on the running messages window In order to send message to this window : 1. Send log with LogMode : MainLogAndMessageTrace MainLogAndProcessLogAndMessageTrace

  1. Directly to this window using the static MessagsRouter.ReportMessage(string sender, string message)

The featurs of the window : 1. Present all messages received 2. Filter the messages according to the sender/s 3. Filter the messages according to one or multipile strings (and or or action between them)
