Detailed Description

An internal event.

Description.
  • Events are actions to perform when conditions are satisfied.
  • There are 2 kinds of events
    1. An internalEvent which is implemented by method
    2. A BaseAlgorithmEvent which is implementing a send of message of the base algorithm
  • The events has 2 parts
    1. The condition which is held in the Event. The role of the condition is to decide whether to perform the event
    2. The action which is implemented by the inherited classes that specify the action to perform
  • This class has 2 roles:
    1. Holds the names of methods that has to be activated when an Event occurs
    2. Holds all the methods needed for the performing of the InternalEvent
