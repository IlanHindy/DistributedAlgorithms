◆ SendMessageString()

private string DistributedAlgorithms.AttributeDictionary.SendMessageString ( object  key)
private

Sends a message string.

Description.
  • This method generate a code that can be used to send a message
  • The method generated has the following parameters:
    1. A dictionary that holds all the fields
    2. The way the list of target ids will be handled:
      1. SelectingMethod.Include - The list of ids are the ids that the message will be sent to
      2. SelectingMethod.Exclude - The message will be sent to all the neighbors but the ons in the ids list
      3. SelectingMethod.All - Send a message to all the neighbors
    3. ids - a list of ints with the ids to include or exclude.
Algorithm.
Usage Notes.
The following is the ways a call can be made for this method: ~~~{.cs} //Include option public void SendAd(dictionary, SelectingMethod.Include, List<int> ids = new List<int> {1,2,3})

//Exclude option public void SendAd(dictionary, SelectingMethod.Exclude, List<int> ids = new List<int> {1,2,3})

//All option public void SendAd(dictionary, SelectingMethod.All}) ~~~.
