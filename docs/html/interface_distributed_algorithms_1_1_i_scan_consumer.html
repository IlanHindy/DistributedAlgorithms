Detailed Description

Interface for scan consumer.

  • The scan consumer is a class that asks for all the attributes of a network element or attribute dictionary
  • The process of activating the scanning is as follows:
    1. Declare a stack in the class
    2. Activate the ScanAndReport method of the object you want to scan
    3. Implement the methods of this interface
