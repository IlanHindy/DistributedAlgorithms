Detailed Description

The syntaxHighlight is a control which contains a text which is syntax highlighted.

Description
The following is the way that this class works:
  1. The class gets a text to highlight and a list of class names
  2. The class divide the text into text fragments in the following way
    1. Find all the text fragments that begins with # and end with eol (the regions)
    2. Find all the text fragments which begins with " and end with "
    3. Find all the text fragments that are c# reserved words
    4. Find all the text fragments which contain a primitive type names
    5. find all the rest of the textFragments
  1. Create an ordered list according to the text begin index while resolving coalitions
  2. Put the text fragments one after the other in the text block
