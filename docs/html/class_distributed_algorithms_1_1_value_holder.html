Check members.

An IValueHolder extension method that check members.

Description.
This is a method that recursively check the members of a tree under the IValueHolder
Algorithm.
  1. Check the members of the IValueHolder:
    1. The network (Has to be the same as the parent)
    2. The permissions (Has to be the same as the parent)
    3. The parent
  2. Cal the CheckMembers of the IValueHolder to implement the recursive check

Equals to.

Description.
  • This method recursively checks if the first IValueHolder is equal to the second
  • The compare is for the values and not the pointers of the IValueHolder
  • The parameter checkNotSameObject is for checking that there was a copy (means created new objects) and not assignments.
Algorithm.
  1. Common checks for all the IValueHolder
    1. The type of the 2 IValueHolder is the same
    2. If checkNotSameObject check that the objects are not the same objects
  2. Activate the recursive compare by calling the IValueHolder's EqualsTo method.

An IValueHolder extension method that gets key string.

Description.
The key string is a string composed from all the keys of the parents chain.
Algorithm.
  1. The method starts with the value holder
  2. While not reached the main NetworkElement (it's parent is null)
    1. Get the key of the value holder in the parent (actually it get's the key of the attribute that is the parent of the value holder)
    2. jump 2 value holders (the attribute and the parent of the attribute)
