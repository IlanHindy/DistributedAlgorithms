public static void DistributedAlgorithms.SyntaxHighlight.TextFragment.HandleCollitions ( List< TextFragment >  list,
TextFragment  item 
)
static

Handles the collisions.

Description.
  • When inserting a new TextFragment a collisions with all the other text fragments has to be resolved
  • The following are the options
    1. If the new Text has no collisions with the existing texts - insert it
    2. If the list has a text that contains the new text - throw the new text
    3. If the new text contains texts in the list - remove the texts from the list
    4. In all the other options (partly collisions) throw the new text
